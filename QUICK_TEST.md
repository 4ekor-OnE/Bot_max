# Быстрая инструкция по тестированию

## 1. Узнайте свой max_id

Отправьте боту `/start` и проверьте логи:
```bash
tail -f /tmp/bot.log | grep "MAX_ID"
```

Вы увидите строку: `📝 ВАШ MAX_ID ДЛЯ НАСТРОЙКИ РОЛИ: 21447227`

## 2. Настройте роль SUPPORT

```bash
cd /root/max_support_bot
source ../venv/bin/activate
python3 test_support_setup.py
```

Введите ваш max_id (или нажмите Enter для тестового).

## 3. Проверьте функции

1. Откройте бота → `/start` → Должно появиться меню специалиста
2. Нажмите "🆕 Новые заявки" → Выберите заявку → "✅ Взять в работу"
3. Нажмите "⚙️ В работе" → Выберите заявку → "✅ Решить" или "⏸️ Отложить"
4. Нажмите "🏪 По магазинам" → Выберите магазин → Просмотрите заявки
5. Нажмите "📊 Моя статистика" → Проверьте метрики
